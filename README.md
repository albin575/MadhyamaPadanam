## 📖 Content Overview

### 🏛️ 1.1 — ആദ്യകാല മാധ്യമങ്ങൾ
> *"ഭാഷയുടെ ആവിർഭാവം മുതൽ ബഹുജന മാധ്യമം വരെ"*

- 🗣️ ഭാഷയുടെ വികാസം (ശബ്ദ → ചിത്ര → എഴുത്ത് → അച്ചടി → ഇലക്ട്രോണിക്)
- 📰 അച്ചടിയുടെ ചരിത്രം (ഗുട്ടൻബർഗ് → ഇന്ത്യ → കേരളം)
- 📄 കടലാസിന്റെ ചരിത്രം (ചൈന → യൂറോപ്പ് → ഇന്ത്യ)
- 📰 മലയാളത്തിലെ പ്രമുഖ പത്രങ്ങൾ (രാജ്യസമക്ഷം → മാതൃഭൂമി)
- 📰 വാർത്തയുടെ ഘടന, ഉറവിടങ്ങൾ, അടിസ്ഥാന തത്വങ്ങൾ

### 🎙️ 2.1 — റേഡിയോ പ്രക്ഷേപണം
> *"മാക്സ്‌വെൽ മുതൽ ആകാശവാണി വരെ"*

- 📡 റേഡിയോയുടെ കണ്ടുപിടിത്തം (മാക്സ്‌വെൽ, ഹെർട്സ്, മാർക്കോണി)
- 🇮🇳 ഇന്ത്യയിലെ AIR / ആകാശവാണി
- 📻 സ്വകാര്യ FM, കമ്മ്യൂണിറ്റി റേഡിയോ
- 🎭 റേഡിയോയും കലാ-സാഹിത്യ മേഖലയും

### 📺 2.2 — ടെലിവിഷൻ
> *"നിപ്കോ മുതൽ 4K ടിവി വരെ"*

- 📺 ടിവിയുടെ ചരിത്രം (ബെയർഡ്, ഫാർണ്സ്വർത്ത്)
- 🇮🇳 ഇന്ത്യയിലെ ടിവി (ദൂരദർശൻ → സ്വകാര്യ ചാനലുകൾ)
- 📊 ടിവി റേറ്റിംഗ് (TRP, BARC)
- 🌐 വിവരസമൂഹം (Information Society)

### 💻 3.1 — നവമാധ്യമങ്ങൾ
> *"ഇൻ്റർനെറ്റ്, സോഷ്യൽ മീഡിയ, ബ്ലോഗുകൾ"*

- 🌐 ഇൻ്റർനെറ്റിന്റെ ചരിത്രം (ARPANET → WWW)
- 📱 സോഷ്യൽ നെറ്റ്‌വർക്കിംഗ് (Facebook, Twitter, Instagram, YouTube, Telegram)
- ✍️ ബ്ലോഗുകൾ, വ്ലോഗുകൾ
- ⚖️ നവമാധ്യമങ്ങളുടെ സാമൂഹിക സ്വാധീനം (നല്ലത് & മോശം)

### ⚖️ 3.2 — സൈബർ നിയമങ്ങൾ
> *"IT Act, DPDP Act, സൈബർ കുറ്റകൃത്യങ്ങൾ"*

- 🔓 തുറന്നെഴുത്ത്, കാംപയിനിംഗ്, വീക്കിലിക്സ്
- 🔒 സ്വകാര്യത, പൊതുമണ്ഡല നിർമ്മിതി
- 😈 ട്രോൾ, ഹാഷ്‌ടാഗ് ആക്ടിവിസം
- ⚖️ IT Act 2000, Amendment 2008, DPDP Act 2023
